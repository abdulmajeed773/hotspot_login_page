معلومات حول برنامج الدخول التلقائي

• تسهيلا منا للدخول على الشبكة تم توفير الدخول على الشبكة عبر برنامج الدخول التلقائي للشبكة والذي يتوفر فية الاتي :

1- عبر البرنامج في خيار ( إدخال الكرت تلقائيا ) يتم الدخول الى الشبكة عن طريق مسح ( تصوير ) شفرة البار كود الموجودة على الكرت

2- ثانيا عبر خيار ( الدخول بكرت سابق ) والذي يوجد فيه اخر عشرة كروت تم الدخول بها عبر البرنامج تستطيع اختيار الدخول على اي كرت منها

اي شخص يستخدم برامج الإختراق سيتم حظره من الشبكة كليا مع الملاحقة القانونية
